next.js + micro cmsで構築したうえむーのポートフォリオサイトです。

2020年に今まで学習した、Three.js / React.js / Next.js / Micro cms などをこのポートフォリオサイトでぶつけてきました。
まだまだ不備なところがあり、さらにアップデートをしみんなが驚くようなサイトを作成することを目指します。

NU stack は 個人の名前のイニシャルとフルスタックを組み合わせたものです。

ロジカルな段取り・設計し、柔軟に対応する人になり、
フロントエンド・サーバーサイド・インフラエンジニアなど様々なスキルを吸収し、
一流のエンジニアになりたいという気持ちを込めてこのサイトを作成しました。

まだまだスキルは全然乏しいですが応援していただけると嬉しいです！
